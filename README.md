# Edu-Map
## Equipo 200 OK
### Descripción
El proyecto consiste en una aplicación interactiva para múltiples materias que utiliza el aprendizaje vivencial y la geolocalización, pensando en que en futuras versiones se podría incluir la realidad aumentada.

El objetivo principal es mejorar el proceso de aprendizaje del usuario ofreciéndole servicios de utilidad basados en su ubicación actual. La aplicación simula una visita remota a un lugar (como un museo, una escuela o cualquier otro sitio), permitiendo al usuario consultar información de manera virtual. Además, busca proporcionar una experiencia inmersiva que permita a los usuarios presentes físicamente llevarse un "recorrido virtual" a casa para seguir consultando información extra, con opciones de búsqueda por filtros.

La solución es altamente transferible, ya que puede utilizarse en múltiples materias simultáneamente (como arte, matemáticas o letras) al aprovechar sitios y contenidos que alberguen información relacionada con distintas disciplinas. Puede emplearse en la escuela, en el hogar o incluso mientras se realiza ejercicio (utilizando la voz), funcionando como un juego de aprendizaje.

El proyecto espera tener un impacto social al ampliar el acceso a nuevos escenarios de aprendizaje sin la necesidad obligatoria de viajar, y un impacto educativo al ofrecer una alternativa detallada y más económica que los viajes, facilitando una forma de aprendizaje más cómoda.
